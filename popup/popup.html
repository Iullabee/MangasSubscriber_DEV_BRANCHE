What is MangasSubscriber ?

It is a firefox extension, available on desktop and android, that lets you :
  • follow mangas on different websites
  • track you reading progress
  • find new chapters for your followed mangas
It works with Fanfox/Mangafox, Mangahere, Webtoons, MangaTown, Readmanganato, Isekaiscan and MangaDex.
The permissions you agreed to during installation are used to :
  • follow a manga (tabs)
  • retrieve new chapters (access to data on [websites])
  • bypass the age restriction on followed mangas (cookies)
  • store data about followed mangas and reading progression (storage)
  • save your list as a file (downloads)
  • save your list online (access to paste.ee)
  • copy error messages from the update console (clipboard)
Please report bugs or send suggestions to mangassubscriber@gmail.com, thank you !
Below is a description of the different parts of the extension, what they enable you to do and how to use them.

The Browser Action is your entry point :

The Icon

  • On the icon is displayed the number of mangas that have at least one chapter you have yet to read (no matter how many unread chapters a manga has, it will be counted as one).
  • When you click it, it will bring up the popup menu.
  • It is unavailable on firefox for android !

The PopUp consists of different parts :

The first part is the Menu

  • add the manga to your list. It can also be : (already in your list), or (not a manga on a supported website)
  • open/close the options panel.
  • open/close the console panel.
  • open/close the help panel (this one).
  • open/close the search panel.
  • version of the extension. close any opened panel.

The second part filters the list and apply actions to all visible mangas

  • hide all mangas in the list whose title doesn't contain what is typed inside.
  • toggle the display of fully read mangas.
  • toggle the display of mangas with unread chapters.
  • toggle the display of mangas with the corresponding tag.
  • refresh the list.
  • The subsequent buttons apply to all visible ( = not filtered) mangas : open selected chapter in a new tab, update, mark all chapters as 'read', / toggle update on/off, remove from the list

The third (and last) part is the list

  • Each line represents a manga (name, date of last update, number of unread chapters, and the list of chapters, followed by the open selected chapter at the end).
  • Hovering over a manga will extend it to show additional options applying to that manga only : remove from the list, select the tags, select the websites, update, mark all chapters as 'read', / toggle update on/off

The Options Panel gives you the following possibilities :

The Options

  • unified chapter numbers : lets you choose whether or not chapters numbers are all formated the same (and volume numbers discarded) regardless of the registered websites numerotation system.
  • performance mode : lets you choose whether or not to trim down the updating process to its minimum.
  • check all sites : lets you choose whether to check only a manga's primary website when looking for new chapters, or all registered websites.
  • navigation bar : will let you navigate between chapters (first, previous, next and last) when reading a subscribed manga. will also unlock a button while reading to mark a chapter as "not read" (if the website can't load images or something).
  • automatically update : will update the list every X hours.
  • limit searches : will only show the first X results when searching on mangas websites.
  • export list to file : will export your list to a local json file.
  • import mangas list from file : will open a webpage to import data saved in a file.
  • export list to paste.ee : will anonymously export your list to paste.ee.
  • import list from paste.ee : will import data previously exported to paste.ee.

The Console Panel shows you what's happening while you're updating :

The Console

  • updating : shows/hides the mangas you have started updating, and on which website. will not show anything when performance mode is enabled.
  • completed : shows/hides the mangas that have successfully been updated. will not show anything when performance mode is enabled.
  • errors : copies (ctrl+c) the errors for easy reporting.

The Search Panel lets you search for random stuff across all available websites :

The Search

  • type something in the search field, and validate by pressing [enter].
  • click the [x] button on the right to clear the search field.

The Results

  • the results (whatever the available websites have found in relation to what you typed) will be shown in a popup.
  • click a website name to hide/show its results.
  • click a manga name to open it in a new tab.
  • click a follow button on the right to follow it directly from the search.
  • click [dismiss] or anywhere outside the popup to close it.
